Edit Colors and Styles

You can edit the Colros and Styles of the theme easily, using SASS. For SASS you may need to use SASS preprocessor application like Scout APP for example.

Start ScoutApp ( or the preprocessor of your choice ) setup your Project, then go to the Theme's scss folder and open style.scss with a code editor to edit the Theme's main styles; The Components can be found in separate files, which have "_" in front.

How to use Google Maps

Since Google have changed their Terms about Google Maps, we have to use Google Maps API in order to display Google Maps on Our website, so if you would like to display the Google Map on your site you will need to add your Google Maps API at the bottom of your main .html ( maybe index.html ), so by default we have a demo API KEY there, so you just have to replace it, as shown below:

- in index.html ( demo ) file at the bottom ( in javascript / scripts section ) you will see this script: 

        <script src="https://maps.googleapis.com/maps/api/js?key=AIzaSyBA40uewXP25u1A4o9u8ueBimZZwIdNLkY"></script>

- in the above script replace the code after js?key= with your own Google Maps API KEY which you can get from here

Setup Contact Form

You can setup the contact form to send emails to your email address in php / form-process.php file edit this code from line 35:

          $EmailTo = "hello@example.com";

Replace the dummy email address hello@example.com with your own email address

In form-process.php you can edit the subject of the emails you will get by editing this code, from line 36:

          $Subject = "New Message Received";

Replace New Message Received from the above code with the subject you want to see when you receive an email from this contact form
